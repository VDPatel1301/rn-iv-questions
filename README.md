# RN_IV_Questions

A simple React Native (Expo) project built with TypeScript to demonstrate:

- A basic calculator screen
- A responsive navbar toggle
- A Two Sum challenge with dynamic input and delete support

## 🔧 Tech Stack

- **React Native**
- **Expo SDK 53**
- **TypeScript**
